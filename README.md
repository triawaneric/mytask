<h2>Features</h2>
</hr>
<ui>
  <li><p>Create new tasks with title, due date, color </p></li>
  <li><p>Move tasks across different stages of your workflow (To Do, Progress, Done)</p></li>
  <li><p>Edit, Delete existing tasks</p></li>
</ui>
